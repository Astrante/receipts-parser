**Правило:** Никогда не добавлять подпись Co-Authored-By в git коммиты.

**Why:** Пользователь явно запретил использование Co-Authored-By подписей.

**How to apply:** При создании git коммитов через Bash tool использовать только commit message без Co-Authored-By строки.
